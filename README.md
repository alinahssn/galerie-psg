Objectif : Créer une galerie interactive dédiée au Paris Saint-Germain et à la photographie sportive pour mettre en valeur des contenus multimédias. 

Réalisation : Conception d'une interface fluide et visuelle permettant une navigation thématique entre les différentes catégories de photos. Intégration et optimisation de contenus vidéos haute résolution avec gestion des paramètres de lecture automatique. Développement réalisé en HTML5, CSS3 et JavaScript. 
Déploiement et hébergement du site via GitHub Pages.
